## Controls
- **LEFT and RIGHT ARROW:** control left and right flops
- **SPACE:** Launch ball

## Debug tools
- **F1:** Activate debug mode / View colliders
- **1:** Spawn ball on the mouse position
- **Left Click:** Grab ball
